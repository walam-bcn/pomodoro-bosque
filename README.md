# 🌳 Pomodoro Bosque

Un temporizador Pomodoro minimalista donde **cada sesión de enfoque completada planta un árbol** en tu bosque del día. Pensado para concentrarte sin distracciones y ver crecer tu progreso.

## ✨ Características

- ⏱️ Temporizador con anillo de progreso animado.
- 🌳 Bosque que crece: cada sesión completada planta un árbol al azar.
- ⚙️ Duraciones configurables de enfoque (1–90 min) y descanso (1–30 min).
- 📊 Estadísticas del día: sesiones, minutos y árboles.
- 🔔 Aviso sonoro suave al terminar cada sesión (sin archivos externos).
- 💾 Guardado automático en el navegador (`localStorage`); no necesita conexión ni servidor.
- ⌨️ Atajo: la **barra espaciadora** inicia y pausa el temporizador.

## 🚀 Cómo usarlo

No requiere instalación ni dependencias. Basta con abrir el archivo:

1. Haz doble clic en `index.html`, o
2. Ábrelo desde tu navegador con `Archivo → Abrir`.

## 🧑‍💻 Uso

1. Ajusta, si quieres, los minutos de **enfoque** y **descanso**.
2. Pulsa **Iniciar** (o la barra espaciadora) para arrancar la sesión.
3. Al completar una sesión de enfoque se planta un árbol 🌳 y empieza el descanso.
4. Usa **Limpiar** para reiniciar el bosque, las sesiones y los minutos del día.

## 🛠️ Tecnología

HTML, CSS y JavaScript puro (vanilla), todo en un único archivo `index.html`. Sin frameworks ni dependencias.
